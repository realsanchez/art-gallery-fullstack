## How to Run the Project

### Backend

Navigate to the backend folder:
``` bash
cd Backend
npm install
npm run dev
```
The backend will run on:
http://localhost:3000

### Frontend

Navigate to the frontend folder:
``` bash
cd Frontend
npm install
ng serve
```
The frontend will run on:
http://localhost:4200
